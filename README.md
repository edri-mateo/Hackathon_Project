# Syllabus Fetch 
## Overview 

Syllabus Fetch is an application that allows you to enter details about
school assessments (date, name, final grade weight, etc.) and generate a `.csv`
file that is compatible with [Google Calendar](https://calendar.google.com/).

## Running 
1. Ensure `javaswing` is installed
2. Navigate to your IDE and go to `src/main/java/org/example/` and open `UI.form`
3. Click the run button

## Usage 
1. Begin by clicking "Add Assessment"
2. Select your assessment type
3. Enter your assessment details 
4. Click "Enter" to add your assessment to the `.csv` file
5. Repeat steps 2-4 for every assessment
6. (Optional): Click the "X" icon to delete an assessment entry
7. Click "Generate .csv" when ready to generate a `.csv` file
8. Select and enter the directory you want to save your `.csv` file to 

## Importing into Google Calendar 
1. Open [Google Calendar](https://calendar.google.com/)
2. Ensure that you are logged into your account
3. Go to your settings
4. Click "Import & export"
5. Select your `.csv` file
6. Click "Import"

_You can import your `.csv` file to any calendar app_
